# 聊聊前端学习

## 1.核心技术栈

```javascript
  极客营开设的前端班,课程体系非常全面,涉及到前端的方方面面,会学习非常多的技 术,html,css,javascript,jquery,Nodejs,Vue,React等等.对于学习能力强的同学来说是一件好事,在几个月时间可以掌握几乎前端所有的知识,毕业后工资立马就可以过万;但是对于大多数同学来说,可能会是一个负担,有的同学部分技术听不懂或者学不好产生忧虑,担心自己找不到工作,比如说有的同学学习到Nodejs就感觉到力不从心.
  其实完全没必要担心这些,因为大部分学习的内容工作和面试中都不会用到!!!既然工作中用不到为啥我们还要开设这些课程呢?因为学习这些可以帮你们更好的理解前端的技术体系,比如说我们学习的Nodejs,工作中几乎用不到,学习的目的就是帮你们了解服务端开发.
  说了这么多到底哪些技术才是我们前端工程师必备的最重要的技术栈呢?
  ## 其实只有两个:javascript和Vue(Vue是javascript框架)
  如果你真的没有精力学习那么多,请务必把时间分配给 javascript和Vue!!!
```

## 3.学习方法

```javascript
1.一定不要看代码!!!一定不要看代码!!!一定不要看代码!!!重要的事情说三遍,再简单的代码都要自己敲出来,你敲的过程会出现很多错误,看代码是永远不会出错的,看懂了你也敲不出来,不要学赵括纸上谈兵,习总书记的名言:空谈误国,实干兴邦!大家要谨记!
2.学习过程中遇到不懂的名词多去百度找答案,看别人写的技术博客,一定要把弄懂,日积月累,你就成为技术大牛了,比如说我现在问你js原型对象是什么?你是不是没学过,是不是可以百度去找答案,原型对象弄懂了你就会掌握一大片js的技术,比如说继承.
3.有时间可以看一些技术书籍和技术视频,js书籍推荐 javascript高级编程,学习视频推荐B站的一套"计算机科学速成课",可以帮你详细的了解计算机的发展历史,建议看三遍以上并做笔记对于你们学习编程和了解计算机非常有帮助,视频地址:https://www.bilibili.com/video/BV1EW411u7th?spm_id_from=333.999.0.0
javascript高级编程电子书籍也在课件中一并给你们了
4.遇到不懂的知识多查技术网站,如MDN Web Docs:https://developer.mozilla.org/zh-CN/
```

## 4.程序员职业的优缺点

```javascript
# 优点
  1.高薪资,薪资随便过万
  2.专注于技术,和代码打交道多,和人打交道少,如果你的人际沟通能力比较弱的话,选这行没错!
  3.行业前景广阔,在互联网时代,未来一定是技术的天下,未来属于你们!
# 缺点
  1.互联网行业普遍有加班的现象,进入这一行要做好加班的准备.
  2.上班后公司性别比较单一,大部分互联网公司都是男性多,可能单身的同学找对象的机会不多.
  3.程序员给人不洗头不洗澡脏的印象,这是部分不爱干净的程序员导致的坏印象,希望大家还是要注意个人卫生,净化我们程序员的邋遢风评!
```
